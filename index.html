    3D漫画スタジオとは

    「3D Manga Studios」は、Blenderを用いて効率的に漫画のようなシーンを表現するシステムです。

    トーン・アウトライン・ライティング・コレクションなど、漫画的なレンダリングを簡単に設定できます。

    特徴

    • 3D Manga Collection Generator：漫画のシーンを作成するために効果的なコレクション（アウトライナー）構造を瞬時に生成
    • Light Collection Tool：専用のマテリアルに対応する３つのライトセットを瞬時に生成
    • Manga Line System：3Dモデルの輪郭を抽出して漫画のような輪郭線を生成
    • Manga Tone Shader：キーライト、フィルライト、バックライトの3つのライトによって構成される影のある専用のマテリアルと追加アセットに格納されているスクリーントーンのような調整可能なマテリアルによる漫画のような質感を再現

    構成

    「3D Manga Studios」アドオンの構成

    • 3D_Manga_Studios/ — アドオン本体
    • Manga Tone Shader Assets/ — マテリアルの素材集

    ドキュメントサイトの構成

    • docs/ — ドキュメント一式（このファイル含む）
    • assets/ — 画像・サンプルファイル
    • mkdocs.yml — サイト設定ファイル

    使い方

    1. インストール

    1. Blenderの「編集 > プレファレンス > アドオン」へ
    2. 「インストール」から .zip を選択して追加
    3. 「3D Manga Studios」を有効化

    2. アセットブラウザへ専用マテリアルの登録

    1. Blenderの「編集 > プレファレンス > ファイルパス」へ
    2. 「ファイルパス」から「アセットライブラリ」にて 右側にある 「＋」ボタン をクリック
    3. 同梱しているManga Tone Shader Assets.zip を解凍し、without shaders に収録されているJP（日本語表記用）、ENG（英語表記用）から１つのフォルダを選択し登録、 設定を保存してから閉じます
    4. アセットブラウザのエリアを追加し、アセットライブラリをwithout shaders に指定する

    3. 3D Manga Studios / 3D漫画スタジオ

    1. 3D漫画コレクションを生成のパネルにある3D漫画コレクションを作成のボタンを押す
    2. オブジェクトを選択しマテリアルとアウトライン生成のパネルでManga Tone Shaderを適用、アウトラインを作成のボタンを押して専用のマテリアルやアウトラインを適用する
    3. ライトコレクション管理のパネル内の追加のボタンを押して専用のライトコレクションと3つのライトとライトの向きを一括操作するエンプティを追加
    4. Manga Tone Shaderのパネルにあるプロパティを操作

    4. マテリアル設定

    • 各オブジェクトに「Manga Tone Shader」を適用
    • シェーダーエディタでトーン濃度・線数を調整可能
